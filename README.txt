# Mowpics

This repository is for a Pinterest-like platform designed to identify and remove AI-generated photos through community voting and automated analysis tools.

# Features Overview

> User Voting System: Allows users to vote on photos suspected of being AI-generated. Sufficient votes will flag the photo for removal.

> AI Detection on Upload: Implements algorithms to analyze photos during upload for signs of AI generation.

> Photo Moderation Dashboard: Tools for administrators to review flagged content and take appropriate action.

> Secure and Scalable: Designed to ensure user data privacy and handle large-scale photo uploads.
